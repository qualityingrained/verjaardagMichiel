Riddle Lights

Solve each riddle and set the lights. Find the correct combination to unlock the secret.

1

"I'm the lone survivor when division pairs dissolve, The seed of every sequence all mathematicians solve. Remove me from a mirror and the world stays much the same— Which number hides beneath this claim?"

2

"I'm the count of choices left when five have wandered past, The tally on a cube's far face when one is shown at last. Lightning's fork may split in threes, but mine are deeper tricks— Which number hums between the flicks?"

3

"A harmony of angles where the sum refuses bends, The smallest crowd required for a cycle that never ends. In fairy tales, good wishes granted often come in me— Which number forms this trinity?"

4

"I'm the quorum of fingers that can cast a single vote, The petals on a blossom children sometimes idly note. Ancient poets carved my shape into celestial guides— Which number in the heavens hides?"

Hold a column to view the riddle. Use the button to set it.
